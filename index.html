🌳 Finsbury Park Loop
~3.2km · Flat tarmac · Parkrun route
Head north on Lennox Road → enter park at Oxford Road gate → follow anti-clockwise tarmac path → one loop = 2.4km, add a half lap for 3.2km.

🌿 Parkland Walk
~4km out & back · Traffic-free · Nature reserve
Walk to FP station → pick up Parkland Walk trail → run north-east along old railway line → turn at 2km near Crouch Hill → run back.
